# Friends Together

Friends Together is an invite-only social space built for close friends.
It helps your circle share what they watch, read, play, and think in a private, community-first environment.

Live: https://friendstogether.com.tr

## Highlights

- Invite-only onboarding
- Real-time social feed
- Media sharing across movies, series, books, games, anime, music, and software
- Quick text posts, likes, and comments
- Community member profiles
- Shared chat area
- Admin tools for invite and member management

## Product Vision

Friends Together is designed to feel more personal than public social platforms:

- Private by default
- Fast and mobile-friendly
- Clean, cinematic interface
- Built around friend groups, not algorithms

## Tech

- Next.js + TypeScript
- TailwindCSS + shadcn/ui
- Firebase Authentication + Firestore
- GitHub Pages deployment
